# Personality Compatibility Plan

작성일: 2026-05-11
브랜치: `feature/personality-compatibility`

## 목표

성향 궁합 기능은 사용자가 어려운 명리 용어를 공부하지 않아도, 두 사람의 관계 흐름을 바로 이해할 수 있게 만드는 기능이다.

핵심은 "누가 맞다/틀리다"가 아니라 아래 질문에 답하는 것이다.

- 우리는 왜 자주 부딪힐까?
- 상대는 어떤 방식으로 편해질까?
- 내가 조심해야 할 말투나 행동은 무엇일까?
- 관계를 좋게 만들려면 오늘 무엇부터 바꾸면 될까?

## 유지할 원칙

- 기존 사주 계산 엔진과 생년월일/성별/출생시간 입력 구조는 유지한다.
- 사주 명식, 오행, 십성, 합충 같은 전문 근거는 내부 판단 자료로 사용하되 본문 전면에 반복 노출하지 않는다.
- 결과 화면은 모바일 기준으로 한 화면씩 읽기 편한 카드 흐름을 우선한다.
- 결제/코인/보관함/재열람 구조는 기존 entitlement 정책을 따른다.
- 기존 궁합 기능을 무리하게 삭제하지 않고, 성향 궁합 레이어를 추가하는 방식으로 진행한다.

## 사용자 흐름

1. 홈 또는 궁합 메뉴에서 `성향 궁합` 진입
2. 내 정보 선택 또는 입력
3. 상대 정보 선택 또는 입력
4. 관계 유형 선택
   - 연애
   - 부부
   - 가족
   - 직장
   - 친구
5. 무료 결과 표시
   - 관계 한 줄 요약
   - 잘 맞는 지점 2개
   - 자주 부딪히는 지점 2개
   - 오늘 바꿔볼 행동 1개
6. 더 깊게 보기
   - 550원 또는 1코인 성향 궁합 상세
   - 990원 관계 대화/갈등 풀이
7. 보관함 저장 및 재열람

## 데이터 입력 구조

필수 입력:

- 내 생년월일
- 내 성별
- 상대 생년월일
- 상대 성별

선택 입력:

- 내 출생시간
- 상대 출생시간
- 내 출생지
- 상대 출생지
- 관계 유형
- 현재 궁금한 문제

시간 미입력 처리:

- 시주 중심 판단은 낮은 신뢰도로 처리한다.
- 결과 본문에는 "태어난 시간이 없어 세부 시간 흐름은 넓게 읽었습니다." 정도로만 표시한다.

## 내부 판단 데이터

성향 궁합에서 활용할 내부 데이터:

- 두 사람의 일간/일지
- 오행 분포 차이
- 강한 오행과 약한 오행
- 십성의 관계 반응
- 합/충/형/해/파 여부
- 관계 유형별 해석 가중치

전면 노출하지 않을 표현:

- 격국
- 용신/기신
- 대운 세부 근거
- evidence_json
- engine/rule version
- 개발자용 메타데이터

## 결과 화면 구조

### 무료 결과

1. 관계 한 줄
2. 우리 관계의 장점
3. 자주 어긋나는 지점
4. 오늘 해볼 행동
5. 더 깊게 보기 CTA

### 상세 결과

1. 관계 온도
2. 감정 표현 방식
3. 돈/생활 리듬
4. 갈등이 커지는 패턴
5. 회복 대화 문장
6. 보관/대화 이어가기

## 상품 구조

무료:

- 성향 궁합 한 줄
- 잘 맞는 지점
- 조심할 지점
- 오늘 해볼 행동

550원 또는 1코인:

- 성향 궁합 상세
- 관계별 실천 문장

990원:

- 갈등/재회/연애 마음 등 문제 중심 풀이

추후 확장:

- PDF 저장
- 대화방에서 관계 질문 이어가기
- 가족 구성원 관계 지도

## 구현 순서

1. 기존 궁합 입력/결과 라우트와 데이터 흐름 확인
2. 성향 궁합 판단 모델 타입 정의
3. 무료 결과 빌더 추가
4. 상세 결과 빌더 추가
5. 궁합 입력 화면에 관계 유형/문제 선택 추가
6. 결과 화면을 간결한 카드 흐름으로 정리
7. 결제 상품 scope_key 연결
8. entitlement 중복 결제 방지 확인
9. 보관함 재열람 스냅샷 연결
10. 모바일 QA

## 완료 기준

- 로그인 사용자와 비로그인 사용자 모두 궁합을 볼 수 있다.
- 저장된 프로필을 클릭해 내 정보/상대 정보를 빠르게 채울 수 있다.
- 결과가 두 사람의 입력값에 따라 달라진다.
- 어려운 명리 용어가 본문에 반복 노출되지 않는다.
- 이미 구매한 상세 궁합은 다시 결제하지 않고 열린다.
- 보관함에서 과거 궁합 결과가 재생성되지 않고 당시 결과로 열린다.
