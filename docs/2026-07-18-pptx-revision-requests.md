## S1. (참고) 사주아이 — 사주풀이 결제 8단계

경쟁사 결제 흐름 캡처. 요지: **정보입력 → 바로 결제 → 결과**. 중간 무료 맛보기 없음.

| 단계 | 화면 |
|---|---|
| 1 | 홈 (무료 수다권 배너 + 유료 메뉴 상단) |
| 2 | 사주 추가 (이름·관계·생년월일·시간) |
| 3 | 입력 내용 확인 |
| 4 | "요즘 어떤 점이 가장 궁금하신가요?" 고민 입력 |
| 5 | 이용권 부족 안내 팝업 |
| 6 | 츄르 가격표 (990 / 2,970 / 4,950 / 9,900) |
| 7 | 주문 확인 및 결제 — **동의 체크박스 1개** |
| 8 | 국내카드 결제 |

## S2. 진단 2가지

1. **유료/무료가 분리되어 있어야 한다**
   - 사주아이: 돈 주고 보는 사주 ↔ 무료 사주가 분리
   - 간지사주(현재): 9,900원 결제 전에 정보만 입력하면 기본사주를 먼저 보여주고, 더 보고 싶으면 결제
   - → **이때 결제 욕구가 떨어짐**
2. **유료 메뉴가 맨 위에 와야 한다**
   - 사주아이: 무료 메뉴가 하단, 돈 주고 보는 메뉴가 맨 위
   - 간지사주(현재): 무료운세·무료타로(HOT)가 상단, 사주·대운(9,900원)이 그 아래

## S3. 메인 화면

- **헤더 로고 옆 "9,900원" 가격 표기 삭제** ("메인 이거 가격 삭제")
- **무료 메뉴에 제한 넣기 + 간단하게**
  - 4개 무료 카드 문구 교체:

    | 현재 | 변경 (제목 / 부제) |
    |---|---|
    | 무료운세 · 오늘 한 줄 | **간단운세** / 짧은 운세풀이 |
    | 무료타로 · 세 장 뽑기 | **딱 3장 타로** / 지금 급할때 |
    | 꿈해몽 · 꿈 한 단어 풀이 | **한 단어 꿈해몽** / 마음이 찜찜할때 |
    | 대화상담 · 선생님께 묻기 | **질문 하나 대화상담** / 당장 답답할때 |

  - **4개 전부 하루 1번으로 제한**

## S4. 카드 가독성 (50대 이상 타깃)

- 메뉴명 문구 **더 크게, 잘 보이는 색깔로**
  - 근거: "50대 이상은 다 이게(사주아이처럼 큰 글씨) 잘 보인다"
  - 시안: 캐릭터 사진 위 `사 주` 큰 글씨(진한 배경 판) + 노란 배지에 부제
- 부제를 **"이걸 통해서 뭘 얻을 수 있는가"** 로 교체:

  | 메뉴 | 현재 부제 | 변경 부제 |
  |---|---|---|
  | 사주 | 내 사주 풀이 | **불안한 앞날, 준비하기** |
  | 대운 | 10년 큰 흐름 | **인생 반전의 순간** |
  | 택일 | 좋은 날 찾기 | **놓치면 안 될 길일** |
  | 궁합 | 두 사람의 흐름 | **나와 맞는 사람은?** |

## S5. 가격 이벤트

- **9,900원 → 3,300원 이벤트 진행**
- 표시 방식: 원래 가격 `9,900원`에 **취소선**, 그 옆에 `3,300원`

---

## S6. 무료 운세 = 간단한 1장짜리 요약본

- 현재 무료 오늘운세 결과가 너무 김 (요약·점수·일진내역·카테고리별·행운패키지·명식·대운CTA·공유·퍼널·피드백·접힘 추가콘텐츠)
- → **1장짜리 요약본**으로 축소

---

## S7. 결제까지 간단하게

- 돈 주고 보는 메뉴는 **정보입력 → 바로 결제 → 보기**
- **중간 맛보기 필요 없음**
- **결제 페이지 창 단순화** — 위에 상단 불필요함(중복)

---

## S8. 결제 동의 단순화

- 현재: 마스터 체크 + [필수] 4종(이용약관 / 개인정보처리방침 / 환불·청약철회 / 디지털콘텐츠 제공·철회) = 총 5개 체크박스
- 목표: 사주아이처럼 **체크박스 1개**로 통합
  - 문안 예시: "위 주문 내용을 확인하였으며, 회원 본인은 이용약관, 개인정보 이용 및 제공(해외 카드결제의 경우 국외제공), 결제에 동의합니다."
- 원문 질문: "이렇게 짧게 가능?"

---

## S9. 결과 하단 메뉴 개선

- 현재: 텍스트 리스트 (사주 상세 풀이 9,900원 / 궁합 풀이 9,900원 / 멤버십 월정액 / 더 깊이 들여다보기)
- 문제: **하단 메뉴가 헷갈림. 다른 메뉴처럼 보임**
- 목표: **텍스트보다는 이미지** — 썸네일 + 질문형 카피
  - "이런 운세는 어때요?"
  - 사주 — 친구나 가족의 사주도 궁금하다면? / 결과를 공유해줄 수 있어요!
  - 궁합 — 우리 사이는 몇점? / SNS에서 소문난 궁합 맛집
  - 대운 — 대운을 아직 모르신다구요? / 언제 물 들어오는지 한번 확인해보자구요
  - 연도별운세 — 특정 연도의 사주가 궁금하다면? / 내후년 운세는 어떨까?
  - 택일 — 중요한 일일수록 좋은 날에! / Top 3 길일을 골라드려요
