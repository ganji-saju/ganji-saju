# 간지사주 디자인 폰트 시스템

간지사주는 흰 배경, 블랙 텍스트, 핑크 포인트를 중심으로 빠르게 읽히는 모바일 운세 서비스입니다. 폰트는 장식용 명조/바탕을 나누지 않고 **Noto Sans KR 단일 계열**로 통일합니다.

## 1. 기본 원칙

- 모든 화면, 버튼, 입력폼, 카드, 풀이 본문은 같은 한글 고딕 계열을 사용합니다.
- 한자, 간지, 가격, CTA도 별도 장식 폰트로 분리하지 않습니다.
- 제목은 폰트가 아니라 굵기, 크기, 여백, 핑크 포인트로 위계를 만듭니다.
- 모바일에서 줄바꿈이 자연스럽고 시니어도 바로 읽을 수 있는 가독성을 우선합니다.

## 2. 적용 폰트

- primary: `Noto Sans KR` via `next/font/google`
- fallback: `Apple SD Gothic Neo`, `Malgun Gothic`, `system-ui`, `sans-serif`

## 3. CSS 변수

- `--font-body` → 전체 기본 폰트
- `--font-sans` → 전체 UI 폰트
- `--font-mono` → 코드/기술값 표시용 고정폭 폰트

이전 장식용 제목/고전/한자 폰트 토큰과 유틸리티 클래스는 새 UI 기준에서 사용하지 않습니다.

## 4. 사용하지 않는 방식

- 제목만 명조체로 바꾸기
- 한자만 다른 폰트로 강조하기
- 고전 인용용 바탕체를 별도 로드하기
- 버튼/가격/카드마다 다른 폰트 패밀리 지정하기
