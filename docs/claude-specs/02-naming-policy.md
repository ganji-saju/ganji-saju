## 0. 변경 사유 (왜 이 문서가 생겼는가)

### 1차 시도 — "X과" 라벨 (기각됨)

원래 사이트는 오행 라벨을 "결단과(金) / 안정과(土) / 열정과(火) / 시작과(木) / 지혜과(水)"로 표기. 문제: "결단과 기운" → 한국인은 자동으로 "결단(과) 기운" = "결단과 기운 두 가지"로 파싱. 한국어 문법 깨뜨림.

### 2차 시도 — 자연 비유 (기각됨)

대안으로 "새싹의 결(木) / 햇살의 결(火) / 흙의 결(土) / 쇠의 결(金) / 물의 결(水)"을 제안. 그러나 사용자 피드백:

> *"새싹·햇살·흙·쇠·물의 결은 사람들이 잘 못보던 풀이라 원래 풀이로 해주는게 좋을거 같아. 오히려 거부감 들어. '표현의 기운', '생각의 기운'도 전혀 어울리지 않아. 정말 사람들이 알아들을 수 있는 문장으로 풀이가 되어야 하는데 듣도 보도 못한 이상한 해석이 되는 느낌."*

→ **자연 비유와 십성 추상명사화 모두 폐기.**

### 3차 — 최종 결정 (이 문서)

**한국인이 사주 얘기할 때 *실제로 쓰는 단어*를 사용한다.**

추상명사 신조어 만들지 않음. 오행은 "목/화/토/금/수 + 기운"이라는 익숙한 표기. 명리 용어(십성·격국·강약)는 원어 그대로 + 짧은 일상어 설명.

---

## 1. 5가지 핵심 원칙

1. **실제 쓰는 단어** — TV 사주, 명리 책, 유튜브에서 쓰는 표현 사용
2. **추상명사 신조어 금지** — "X의 결", "X의 기운"형 패턴 금지
3. **오행은 "X 기운"** — 목 기운 / 화 기운 / 토 기운 / 금 기운 / 수 기운으로 일관
4. **명리 용어 원어 + 짧은 설명** — 식신·정인·격국·신강 같은 단어는 그대로
5. **한자 노출 최소화** — 사주팔자 8글자 카드에서만, 본문 0개

## 2. 오행 어휘

### 권장 — "X 기운" 표기

| 오행 | 권장 표기 | 사용 예 |
|------|---------|--------|
| 木 | **목 기운** | "이 사주는 목 기운이 잘 발달되어 있어요" |
| 火 | **화 기운** | "화 기운이 약해서 의식적 보강이 필요합니다" |
| 土 | **토 기운** | "토 기운이 많아서 안정감이 큰 사주예요" |
| 金 | **금 기운** | "금 기운을 들이면 단단해집니다" |
| 水 | **수 기운** | "수 기운이 흐름을 만들어줘요" |

### 짧은 일상어 설명 (필요 시 옆에 붙임)

| 오행 | 핵심 의미 한 줄 |
|------|------------|
| 목 기운 | 자라남, 추진, 시작 |
| 화 기운 | 표현, 밝게 퍼짐, 열정 |
| 토 기운 | 담아냄, 안정, 기름 |
| 금 기운 | 단단함, 결단, 마무리 |
| 수 기운 | 흐름, 깊이, 사색 |

본문에서 첫 등장 시: "**금 기운(단단함과 결단)**" 식으로 한 번만 짧게. 이후는 그냥 "금 기운".

### 절대 금지 어휘

- 새싹의 결, 햇살의 결, 흙의 결, 쇠의 결, 물의 결 (자연 비유, 2차 시도)
- 결단과, 안정과, 열정과, 시작과, 지혜과 (X과 라벨, 1차 시도)
- 새싹/햇살/쇠 같은 자연 명사를 *단독으로* 오행 대신 사용
- "목의 기운 / 화의 기운" 형 (조사 "의" 들어간 형태도 어색)

## 3. 십성 어휘

### 권장 — 원어 + 짧은 설명

명리 용어는 사람들이 사주 보러 가서 *그대로 듣는 단어*. 굳이 바꿀 필요 없음.

| 십성 | 권장 표기 | 첫 등장 시 설명 |
|------|---------|----------|
| 식신 | 식신 | 표현하고 베푸는 별 |
| 상관 | 상관 | 재능을 드러내는 별 |
| 정인 | 정인 | 돌봄과 배움의 별 |
| 편인 | 편인 | 혼자 깊이 파고드는 별 |
| 정관 | 정관 | 책임과 규범의 별 |
| 편관 | 편관 | 밀어붙이는 별 |
| 정재 | 정재 | 꾸준한 살림의 별 |
| 편재 | 편재 | 움직이는 돈의 별 |
| 비견 | 비견 | 같이 가는 별 |
| 겁재 | 겁재 | 경쟁하며 끌어가는 별 |

### 사용 패턴

- 첫 등장: "**식신(표현하고 베푸는 별)**이 잘 발달되어 있어요"
- 이후: "식신이 강해서…"
- "별"이라는 표현은 사주에서 일반적이라 거부감 없음

### 절대 금지 어휘

- 표현의 기운, 생각의 기운, 절제의 기운, 직관의 기운 (추상명사화)
- 돌봄의 기운, 베푸는 기운 같은 형용 추상명사
- 표현하는 결, 돌봄의 결 같은 "X의 결" 패턴
- 표현형 사주, 돌봄형 사주 같은 신조어 형용사

## 9. 핵심 키워드 단어 — "결" 단어 제한

### "결" 단어 사용 정책

"결"은 한국어에서 *옷감 무늬, 나무 결, 살결* 같은 데 쓰는 단어. **사주 얘기에 일상적으로 쓰는 단어가 아님.** 

이전 스펙에서 "결단의 결", "타고난 결" 같이 *비유적*으로 쓰려고 했는데, 사용자가 "거부감 든다"고 명시. 

규칙:

| 위치 | "결" 사용 |
|------|--------|
| 한 줄 요약 | ❌ 금지 |
| 점수 라벨 | ❌ 금지 (5단계 라벨 모두 "사주"로) |
| 평생 활용 카드 제목 | ❌ 금지 |
| 본문 단락 | ⚠️ 단락당 최대 1회. *정말 필요할 때만* |
| 카드 부제·본문 | ❌ 금지 (대안 단어 사용) |

### 대안 어휘

- **사주** (가장 무난, 가장 익숙)
- **성향** (성격·기질)
- **기운** (에너지)
- **흐름** (시기·변화)
- **자리** (위치·역할)
- **방식** (방법·접근)
- **타입** (성향 분류 시)

### 예시 비교

| BEFORE (어색) | AFTER (자연스러움) |
|------------|---------------|
| 잘 다듬어진 **결** | 강점이 명확한 **사주** |
| 타고난 **결**은 물처럼… | 타고난 **성향**은 물처럼… |
| 부드럽게 흐르는 **결** | 부드럽게 흐르는 **성향** |
| 단단함의 **결**이 부족 | 단단함의 **기운**이 부족 |
| 본인 **결**이 약한 편 | 본인 **기운**이 다소 약한 편 |

### 한자 매핑 점검 (별도 검증 필요)

진단서에서 이미 짚었던 `己 → 기` (근 X) 오류 외에도, 10천간·12지지 매핑 전수 검사 권장:

- 甲=갑, 乙=을, 丙=병, 丁=정, 戊=무, **己=기**, 庚=경, 辛=신, 壬=임, 癸=계
- 子=자, 丑=축, 寅=인, 卯=묘, 辰=진, 巳=사, 午=오, 未=미, 申=신, 酉=유, 戌=술, 亥=해

## 11. 위치별 BEFORE / AFTER

### 11-1. 점수 5단계 라벨

| 점수 | BEFORE (구) | AFTER (정책) |
|------|-----------|----------|
| 90~100 | 균형 잘 잡힌 결 | **균형이 잘 잡힌 사주** |
| 75~89 | 잘 다듬어진 결 | **강점이 명확한 사주** |
| 60~74 | 흐름이 무난한 결 | **흐름이 무난한 사주** |
| 45~59 | 자기 관리가 빛나는 결 | **자기 관리가 빛나는 사주** |
| 0~44 | 보강의 여지가 큰 결 | **보강의 여지가 큰 사주** |

### 11-2. 오행 차트

**BEFORE**:
```
새싹  ▓▓░░░░░░░░  목  2개
햇살  ▓▓▓▓░░░░░░  화  4개
흙   ▓▓░░░░░░░░  토  2개
쇠   ░░░░░░░░░░  금  0개  ⚠ 보강 필요
물   ▓▓▓▓▓░░░░░  수  5개  ⚠ 과다
```

**AFTER**:
```
목 기운  ▓▓░░░░░░░░  2개
화 기운  ▓▓▓▓░░░░░░  4개
토 기운  ▓▓░░░░░░░░  2개
금 기운  ░░░░░░░░░░  0개  ⚠ 보강 필요
수 기운  ▓▓▓▓▓░░░░░  5개  ⚠ 과다
```

### 11-3. 보강 가이드 카드

**BEFORE**:
> ✨ 보강할 기운 — **쇠의 결**: 단단함과 결단
> 체크리스트, 정기 회고 같은 '단단한 구조'가 평생 큰 보강이 됩니다.

**AFTER**:
> ✨ 보강할 기운 — **금 기운** (단단함과 결단)
> 체크리스트, 정기 회고 같은 '단단한 구조'가 평생 큰 보강이 됩니다.

### 11-4. 한 줄 요약

**BEFORE**:
> "조용히 살피고 흐르는 사주, 단단한 기준 하나가 평생을 받쳐줍니다"

**AFTER (그대로 유지 — 이미 정책 부합)**:
> "조용히 살피고 흐르는 사주, 단단한 기준 하나가 평생을 받쳐줍니다"

→ 이미 "사주" 단어 사용, "결" 없음. OK.

### 11-5. 본문 단락 1 (당신은 어떤 사람인가)

**BEFORE**:
> "테스트 님의 타고난 **결**은 물처럼 부드럽게 흐르는 성향이에요. … 표현하고 살피는 **기운**은 잘 살아 있는데, 결단하고 마무리하는 **기운**이 보강을 필요로 해요."

**AFTER**:
> "테스트 님의 타고난 **성향**은 물처럼 부드럽게 흐르는 편이에요. … 표현하고 살피는 **기운**은 잘 살아 있는데, 결단하고 마무리하는 쪽이 보강을 필요로 합니다."

### 11-6. 본문 단락 4 (지금 시기)

**BEFORE**:
> "이 사주에는 **'쇠의 결'—단단한 구조—을 들이는 게 핵심**입니다."

**AFTER**:
> "이 사주에는 **금 기운(단단함과 결단)을 들이는 게 핵심**입니다."

### 11-7. 평생 활용 3가지 카드

**BEFORE**:
> 카드 3 제목: **"쇠의 결을 들이기"**
> 부제: "단단한 구조가 보강"

**AFTER**:
> 카드 3 제목: **"금 기운 들이기"**
> 부제: "단단한 구조가 평생 보강"

### 11-8. 십성 풀이

**BEFORE (어색)**:
> "**표현의 기운**이 강해서, **생각의 기운**이 보강을 필요로 합니다."

**AFTER**:
> "**식신(표현하고 베푸는 별)**이 강해서, **정인(돌봄과 배움의 별)**이 보강을 필요로 합니다."

또는 첫 등장 후 짧게:
> "식신이 강해서 표현력은 좋은데, 정인이 약해서 차분히 정리하는 자리가 부족해요."

## 12. 자동 검증 정규식

이 정책 적용 여부 확인용. 모든 패턴이 **0건**이어야 합니다.

```typescript
const FORBIDDEN_PATTERNS = [
  // 자연 비유 + "의 결" (2차 시도 흔적)
  /(새싹|햇살|흙|쇠|물)의\s*결/g,
  
  // 자연 명사 단독 사용 (오행 대신)
  /\b(새싹|햇살)\s+(기운|결|흐름)/g,
  
  // X과 라벨 (1차 시도 흔적)
  /결단과|안정과|열정과|시작과|지혜과/g,
  
  // 십성 추상명사화
  /(표현|생각|절제|직관|돌봄|관찰|베푸는|밀어붙이는)의\s*기운/g,
  
  // X의 결 패턴 일반 (안전망)
  /[가-힣]+의\s*결[은이를을과와\s]/g,
  
  // X형 사주 신조어
  /(표현|돌봄|재물|관계|기준)형\s*사주/g,
  
  // 추상명사 + 결
  /(돌봄|표현|기준|단단함)의\s*결/g
];

function checkNamingPolicy(text: string): string[] {
  const violations: string[] = [];
  for (const pattern of FORBIDDEN_PATTERNS) {
    const matches = text.match(pattern);
    if (matches) {
      violations.push(...matches.map(m => `금지 어휘: "${m}"`));
    }
  }
  return violations;
}
```

이 검증을 LLM 후처리 단계(`saju-total-review-llm-spec.md` §7) + 사이트 본문 grep에 모두 적용.

## 13. 시스템 프롬프트 추가 규칙

모든 LLM 호출의 시스템 프롬프트에 다음 블록을 추가:

```text
[어휘 정책 — 절대 규칙]

1. 오행은 반드시 "X 기운" 형태로 표기:
   - 목 기운, 화 기운, 토 기운, 금 기운, 수 기운
   - 금지: "쇠의 결", "햇살의 결" 등 자연 비유
   - 금지: "결단과", "안정과" 등 X과 라벨
   - 금지: "표현의 기운", "생각의 기운" 등 십성 추상명사화

2. 십성·격국·강약은 원어 그대로 + 짧은 설명:
   - 첫 등장: "식신(표현하고 베푸는 별)"
   - 이후: "식신"
   - 금지: "표현형 사주", "돌봄의 결"

3. "결" 단어 사용 제한:
   - 한 줄 요약·라벨·카드 제목: 사용 금지
   - 본문 단락: 단락당 최대 1회, 정말 필요할 때만
   - 대안: 사주, 성향, 기운, 흐름, 자리

4. 한자 본문 노출 금지:
   - 본문에 천간/지지/오행 한자 등장 금지
   - 사주팔자 8글자 카드만 한자 허용
```
